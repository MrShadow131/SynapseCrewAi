# SynapseCrewAi
Synapse Crew AI es un MAS (Multi-agent System) que se encarga de desglozar un tema complejo guiandote y ayudandote en el proceso de estudio, desarmando el tema principal en subtemas, generando un indice completo de estudio y hasta otorgando al estudiante una base con documentos y contenido para estudiar con mas comodidad.

##### *¿Que es MAS?*
*MAS significa Multi-Agent System, y si, a lo que se refiere es que un MAS es un entorno donde conviven varios agentes de IA, y claro, el conjunto de estos terminan formando a una IA nueva.
El objetivo de SynapseCrew es justamente poder ayudar a esos estudiante como yo que quieren estudiar de forma autodidacta y a veces pierden mucho tiempo buscando informacion, o el orden mas concreto y preciso, por eso traigo la solucion: un MAS que permita lograrlo.
Claro que como estudiante aun no termino formalizar un contenido completo por eso es que para este proyecto tengo una integracion importante que aclarar: Gemini IA. Obviamente esta solo siendo de ayuda para comprender comandos y opinar sobre mi estructura actual, etc.*

## Funcionamiento
Para poder hacer que esta IA cobre vida utilizo un framework potente y bastante conocido: CrewAI, el cual me permite establecer un entorno con python y desde alli generar mi propia estructura para tener a mis agentes, herramientas, APIs integradas, documentos, funciones personalizadas, entre muchas otras cosas.

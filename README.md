# SynapseCrewAi
Synapse Crew AI es un MAS (Multi-agent System) que se encarga de desglozar un tema complejo guiandote y ayudandote en el proceso de estudio, desarmando el tema principal en subtemas, generando un indice completo de estudio y hasta otorgando al estudiante una base con documentos y contenido para estudiar con mas comodidad.

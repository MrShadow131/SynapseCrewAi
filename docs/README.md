# Explicacion de funcionamiento del framewrok _CREWAI_
Bienvenidos a esta carpetita, aqui deje un archivo formato MarkDown para que puedan ver como es el funcionamiento interno del framework, ire yendo desde los archivos 'main.py' y 'crew.py' hasta a como definir agentes y tareas. La idea es hacerlo lo mas claro posible asi entienden como es que funciona este framework.\
Una __IMPORTANTE ACLARACION__ es que esta pequeña guia la hago para poder entender mejor yo, y ustedes, ademas que analizo los proyectos de ejemplo del repositorio 'crewAI-examples' de "CrewAIInc":\
[Acceder al repositorio de ejemplos](https://github.com/crewAIInc/crewAI-examples)\
[Acceder al perfil de la compania CrewAI](https://github.com/crewAIInc)\
Recomiendo fuertemente hecarle una visita a los ejemplos oficiales una vez visto esta guia ya que muestran muchos ejemplos y distintas ejecuciones para cada situacion, entre ellas un generador de landing-pages, un constructor de juegos, estrategias para marketing, entre muchos otros.
